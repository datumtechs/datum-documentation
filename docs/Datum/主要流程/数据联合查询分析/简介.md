## 数据联合查询分析简介

数据联合查询分析模块提供了一系列API服务，可使用已资产化的数据进行联合计算。



### 为什么使用本部分API？

数据联合查询分析可由帮助用户更方便、安全、快速地执行联合计算，因为：

+ 本模块api集成了常见的联合计算场景，如联合查询、联合统计分析、联合模型训练和预测
+ 开发者无需过多了解各种算法的实现细节，可直接调用，降低了使用门槛。



### 本部分API支持哪些联合计算？

+ 隐私求交(PSI) 
+ 隐私标签查询(Labeled PSI)
+ 隐私SQL查询
+ 联合统计分析
+ 联合模型训练
+ 联合模型预测
